# 🧗 Bouldering App (Fullstack)

A fullstack web application where users can track climbing routes, upload images, and monitor their progress.

## 🚀 Tech Stack

### Frontend
- React (Vite)
- Material UI
- React Router
- Axios
- Leaflet (maps)

### Backend
- Node.js + Express
- MongoDB (Mongoose)
- JWT Authentication
- Cloudinary (image uploads)
- Nodemailer (email)
- Joi validation

### Testing
- Vitest
- Supertest

## ✨ Features
- User authentication (login/register)
- Upload and manage climbing routes
- Image upload with Cloudinary
- Interactive map (Leaflet)
- Track climbing progress
- REST API backend

## 🧠 What I Learned
- Building fullstack applications
- Authentication and security (JWT, hashing)
- Handling file uploads and external services
- Structuring scalable backend APIs
- Debugging across frontend and backend
