# 🧗 Bouldering App

A React application where users can track climbing routes, log progress, and explore their bouldering sessions.

## 🚀 Tech Stack
- React
- JavaScript
- Typescript
- HTML / CSS

## ✨ Features
- Track completed climbs
- Filter and search routes
- View climbing progress over time
- Dynamic UI updates based on state

## 🧠 What I Learned
- Managing state in React applications
- Debugging using React DevTools and browser tools
- Working with component-based architecture
- Handling data flow and avoiding prop drilling

## 🔧 Challenges
One of the main challenges was structuring the application and managing state across components.  
I solved this by organizing components more clearly and using state management patterns to keep the data predictable.
